# Floating Particles

Floating Particles 是一个 Halo 前台动效插件，用于为博客页面添加可配置的粒子漂浮效果和鼠标动效。插件可以在 Halo 后台直接开启、关闭和切换效果，不需要修改主题代码。

![后台设置](docs/images/settings.gif)

## 功能亮点

- 支持在 Halo 后台一键开启或关闭前台动效。
- 页面粒子效果和鼠标动效分开配置，可以单独选择。
- 支持调整粒子数量、动效颜色、透明度和速度。
- Three.js 已打包到插件本地静态资源，不依赖外部 CDN。
- 关闭或卸载插件后，不会在主题模板中留下额外代码。

- 星尘漂浮

## 鼠标动效

鼠标动效用于增强访问者在前台页面中的交互体验，可以选择安静轻量的光环、拖尾，也可以选择更明显的点击礼花和点击爱心。

- 点击礼花
- 水波涟漪
- 拖尾光点
- 跟随小星星
- 光标星星跟随
- 点击爱心
- 鼠标光环
- WebGL 光标尾巴

## 使用说明

1. 在 Halo 后台进入插件管理页面。
2. 上传并启用 Floating Particles 插件。
3. 进入插件设置页面。
4. 打开 `开启前台动效`。
5. 在 `粒子效果` 中选择页面动效。
6. 在 `鼠标动效` 中选择鼠标交互动效。
7. 按需调整数量、颜色、透明度和速度。
8. 保存设置后，刷新前台页面查看效果。

## 推荐配置

如果希望长期默认开启，建议选择更轻量、不会遮挡阅读内容的组合：

- 页面粒子效果：星尘漂浮、星空漂浮、萤火闪烁、细雨
- 鼠标动效：拖尾光点、鼠标光环、跟随小星星
- 透明度：`0.35` 到 `0.65`
- 粒子数量：`60` 到 `100`

如果页面文字较多，建议降低透明度，避免同时开启彩纸、流星雨、水波等视觉存在感较强的效果。

## 兼容性

- 支持 Halo `>=2.23.0`
- 支持现代桌面端和移动端浏览器
- WebGL 光标尾巴需要浏览器支持 WebGL

## 常见问题

### 是否需要修改主题代码？

不需要。插件通过 Halo 插件机制向前台注入动效脚本，启用后即可使用。

### 是否依赖外部 CDN？

不依赖。插件所需的前端脚本和 Three.js 文件都已打包到插件本地静态资源中。

### 关闭或卸载后会有代码残留吗？

不会。关闭或卸载插件后，前台动效会停止，不会修改你的主题模板文件。
